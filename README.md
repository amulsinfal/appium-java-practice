# Appium Java Automation Practice

This repository serves as a demonstration of mobile automation testing skills using **Appium**, **Java**, and **Maven**.

## Tech Stack:  
1. Language: Java  
2. Automation: Appium (v2.x)  
3. Build Tool: Maven  
4. Test Runner: TestNG

## Project Structure:  
- **src/test/java/com/solution**: Contains all core test logic and test scripts.
- **pom.xml**: Manages all dependencies and build configurations.

## Key Skills Demonstrated:  
1. **Mobile Gestures:** Automated complex user interactions including Swipes, Scrolls, and Drag-and-Drop actions using the W3C Actions API.
2. **Element Locators:** Implementation of robust locator strategies using Accessibility IDs for speed and XPath for dynamic elements.
3. **App Management:** Full control over the app lifecycle: Installing, Uninstalling, Activating, and Terminating applications during test execution.
4. **Device Interaction Scenarios:** Handling hardware-level events such as Locking/Unlocking the device and changing screen Rotation (Portrait/Landscape).
